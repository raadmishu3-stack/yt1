🚀 Google IDX VM Setup Guide
🖥️ 1. Installing VM in Google IDX
Google IDX: https://idx.google.com/

Sign up on Google IDX.
Create a new workspace using this repository (Do NOT change the repo name):
Repository URL: https://github.com/NotGamerPiie/pie_vps

▶️ VM Setup Command
bash <(curl -fsSL https://raw.githubusercontent.com/NotGamerPiie/idx-vps/main/vps.sh)
🐦 2. Installing Pterodactyl Panel in Google IDX
Run the installer script:

bash <(curl -s https://ptero.jishnu.fun)
☁️ 3. Cloudflare Setup
Sign up on Cloudflare for DNS & SSL setup: https://dash.cloudflare.com/

🎉 Setup Completed Successfully!
Your Google IDX VM + Pterodactyl Panel is now installed.
